# 云数据库集合说明

## 需要创建的集合（5个）

在微信开发者工具 → 云开发控制台 → 数据库 中，依次创建以下集合：

| 集合名 | 用途 | 权限设置 |
|--------|------|----------|
| `users` | 用户信息 | 仅创建者可读写 |
| `todos` | 待办事项 | 仅创建者可读写 |
| `categories` | 分类管理 | 仅创建者可读写 |
| `reminders` | 提醒记录 | 仅创建者可读写 |
| `subscribeQuotas` | 订阅消息配额 | 仅创建者可读写 |

## 导入数据

每个 `.jsonl` 文件对应一个集合的初始测试数据。

导入步骤：
1. 打开微信开发者工具 → 云开发控制台 → 数据库
2. 点击对应集合
3. 点击右上角「导入」
4. 选择对应的 `.jsonl` 文件
5. 冲突模式选择「Insert」
